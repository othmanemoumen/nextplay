About This Game

Red Dead Redemption 2 is an action-adventure game developed by Rockstar Games. It was released in October 2018 for PlayStation 4, Xbox One, and later for PC. The game is set in the late 1800s in a fictionalized version of the American West, a time of significant change as the Wild West era comes to an end.
